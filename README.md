# LCR Convertible Note Sensitivity Dashboard

Interactive dashboard to analyze the impact of convertible note conversion on minority shareholders.

## Features

- **Note Terms Configuration**: Adjust principal amount and interest accrual period
- **Valuation Simulator**: Slide through different valuation scenarios
- **Sensitivity Table**: View impact across multiple valuation points ($20M - $70M)
- **Series A Projection**: Model a future equity round and its dilution impact
- **Waterfall Visualization**: Track ownership evolution through funding events

## Key Terms (Fixed)

- **Pre-Money Cap**: $40M
- **Interest Rate**: 6% annual
- **Discount**: 20%
